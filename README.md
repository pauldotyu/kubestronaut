# Kubestronaut

Workshop for preparing to become a Kubestronaut. Work in progress.

## Getting Started

To run the documentation locally, you can use the following commands:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
mkdocs serve
```
